VEGA DOUA 
------------------------------------------------------------
Résumé du jeu : Déplacement d’un personnage (un dresseur de créatures nommées végamons) dans différentes cartes, 
avec des combats entre végamons, des interactions avec d’autres personnages et des quêtes à accomplir. 
Le but du jeu étant de vaincre le dernier champion d’arène.

Toutes les informations sur comment jouer sont données dans le jeu.
------------------------------------------------------------
Comment lancer Vega Doua : 
Extraire et ouvrir le dossier Vega Doua.
Ouvrir le fichier Accueil.java dans Geany.
Compiler et exécuter le fichier.


-----------------------------------------------------------
Contenu de l’archive :
.vscode -> Fichiers de gestion du projet avec l'IDE Visual Studio Code
Images -> contient toutes les images/graphismes du jeu
Musiques -> contient toutes les musiques du jeu
Autres fichiers -> Fichiers contenant le code du jeu développé avec JAVA
