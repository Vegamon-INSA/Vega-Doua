# Vega-Doua
Projet informatique de 2ème année de l'INSA Lyon.

Reproduction d'un jeu type pokémon en Java.

### Lancement du jeu
Pour démarrer le jeu, il suffit de compiler et d'exécuter le programme Accueil.java dans le dossier Code.
